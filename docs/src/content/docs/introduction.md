**@wdc-ui/ng** is a collection of re-usable components that you can copy and paste into your apps.

## Why?

I wanted a way to use consistent, beautiful components without being tied to a heavy library like Angular Material.

## Features

- ✅ **Customizable:** Source code in your project.
- ✅ **Lightweight:** Zero runtime overhead.
- ✅ **Modern:** Built with Angular Signals & Tailwind.
