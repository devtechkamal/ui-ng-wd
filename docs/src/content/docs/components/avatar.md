---
title: Avatar
description: An image element with a fallback for representing the user.
---

## Installation

```bash
npx @wdc-ui/ng@latest add avatar
```

## Usage

```
<wdc-avatar
  src="[https://github.com/devtechkamal.png](https://github.com/devtechkamal.png)"
  alt="@devtechkamal"
  fallback="DK"
/>
```
